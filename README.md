# 劳务费税费换算工具

第一版轻量网页工具：无登录、无数据库、上传即算、导出即走。

## 1. 功能范围

- 支持上传 Excel 批量计算；
- 支持网页手动录入；
- 支持从 Excel 复制粘贴到网页表格；
- 按 `年份 + 月份 + 身份证号码` 累计；
- 计算过程使用高精度 Decimal；
- 页面展示和 Excel 导出保留 2 位小数；
- 导出 Excel 包含三个 Sheet：
  - `劳务费税费换算台账`：严格贴近原表结构，计算列输出最终数值；
  - `清晰版台账`：按业务阅读习惯重新组织，计算列输出最终数值；
  - `公式版台账`：严格贴近原表结构，K-X 计算列保留 Excel 公式；
- 前两个 Sheet 的计算列只输出最终数值，第三个 Sheet 保留 Excel 公式，方便查看和复核计算来源。

## 2. 输入字段

基础数据至少需要以下字段：

| 字段 | 必填 | 说明 |
|---|---|---|
| 年份 | 是 | 参与累计 |
| 月份 | 是 | 参与累计，1-12 |
| 事业部 | 否 | 原样输出 |
| 省区 | 否 | 原样输出 |
| 报销人 | 否 | 原样输出 |
| 会计 | 否 | 原样输出 |
| 姓名 | 是 | 原样输出 |
| 身份证号码 | 是 | 参与累计 |
| 税后劳务金额 | 是 | 核心计算金额，必须大于 0 |

## 3. 计算规则

计算链条：

```text
单次税后金额
→ 本月累计税后金额
→ 反推累计税前金额
→ 计算累计个税
→ 拆出单次个税
→ 计算单次增值税
→ 计算单次附加税
→ 计算累计含税开票金额
→ 拆出单次应开票金额
→ 计算单次应付款金额
→ 核对
```

### 3.1 税后反推税前

| 累计税后金额 | 反推累计税前金额 |
|---:|---|
| ≤ 800 | 税前 = 税后 |
| ≤ 3360 | 税前 = （税后 - 160）÷ 0.8 |
| ≤ 21000 | 税前 = 税后 ÷ 0.84 |
| ≤ 49500 | 税前 = （税后 - 2000）÷ 0.76 |
| ＞ 49500 | 税前 = （税后 - 7000）÷ 0.68 |

### 3.2 个税

个税按累计金额计算，再拆回单次：

```text
单次个税 = 当前累计个税 - 之前已扣个税
```

累计个税规则：

| 累计税前金额 | 个税规则 |
|---:|---|
| ≤ 800 | 0 |
| 800 - 4000 | （累计税前金额 - 800）× 20% |
| ＞ 4000 | 累计税前金额 × 80% 后按 20% / 30% / 40% 三档计算 |

### 3.3 增值税与附加税

```text
本次对应税前金额 = 单次税后金额 ÷ 本月累计税后金额 × 本月累计税前金额
```

| 本次对应税前金额 | 增值税 |
|---:|---:|
| ≤ 500 | 0 |
| ＞ 500 | 本次对应税前金额 × 1% |

```text
附加税 = 增值税 × 12% × 50% = 增值税 × 6%
```

### 3.4 开票、付款、核对

```text
累计含税金额 = 累计税前金额 + 累计增值税 + 累计附加税
单次应开票金额 = 当前累计含税金额 - 之前已开票金额
单次应付款金额 = 单次应开票金额 - 单次个税
核对 = 税后劳务金额 + 增值税 + 附加税 - 单次应付款金额
```

正常核对值应为 `0.00`。

## 4. 后端启动

进入后端目录：

```bash
cd backend
python -m venv .venv
source .venv/bin/activate  # Windows 使用 .venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

访问健康检查：

```text
http://127.0.0.1:8000/api/health
```

## 5. 前端启动

进入前端目录：

```bash
cd frontend
npm install
npm run dev
```

打开：

```text
http://127.0.0.1:5173
```

## 6. 后端接口

| 接口 | 方法 | 作用 |
|---|---|---|
| `/api/health` | GET | 健康检查 |
| `/api/template` | GET | 下载基础数据模板 |
| `/api/calculate/upload` | POST | 上传 Excel 并计算 |
| `/api/calculate/manual` | POST | 手动数据计算 |
| `/api/export` | POST | 导出最终 Excel |
